# Retail-BI-Dashboard-skill
为快消/零售行业分析师构建标准化 BI 看板的完整指导体系。当用户涉及以下任何场景时必须使用本 Skill：制作数据看板、搭建 BI 报表、设计指标体系、编写看板 PRD、生成 QuickBI 配置方案、分析连锁零售/零食量贩/饮料/乳制品/商超/便利店/社区零售行业数据。即使用户只是说"帮我做个销售看板"或"设计一下指标"，也应立即触发本 Skill。
